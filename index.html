Knowledge Atlas

Dynamically generate an expandable taxonomy across formal sciences, natural sciences, social sciences, and philosophy, then request concept maps and categorized bibliographies for any node in the tree. The app is structured as static files plus serverless API routes, so it fits Vercel directly.

Honesty note: no model can guarantee a perfectly exhaustive taxonomy in one pass. This app is designed to keep expanding, refining, and finding missing siblings so you can push coverage deeper over time without pretending the first pass is final.
